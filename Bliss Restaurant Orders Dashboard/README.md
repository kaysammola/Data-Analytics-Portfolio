# Bliss Restaurant Orders Dashboard

## Project Overview

This dashboard analyzes restaurant operations, customer orders, cuisine performance, delivery times, and food preparation efficiency.

---

## Business Problem

Restaurant managers need operational insights to improve customer satisfaction, optimize delivery performance, and maximize profitability.

---

## Dataset Information

The dataset contains:

- Customer Orders
- Cuisine Types
- Ratings
- Delivery Time
- Food Preparation Time
- Restaurant Performance

---

## Tools Used

- Microsoft Power BI
- Power Query
- DAX

---

## Dashboard Features

- Customer Rating KPI
- Order Analysis
- Cuisine Performance
- Delivery Time Analysis
- Food Preparation Analysis
- Restaurant Comparison

---

## Key Insights

- Average customer rating reached 2.99.
- Indian and Italian cuisines generated the highest average order value.
- Weekdays experienced longer preparation times.
- Burger Barn and Pizza Place generated the highest order revenue.
- Weekend deliveries were slightly slower.

---

## Business Recommendations

- Improve customer service quality.
- Reduce food preparation times.
- Optimize delivery logistics.
- Promote top-performing cuisines.
- Continuously monitor customer feedback.
