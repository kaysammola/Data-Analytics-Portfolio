# Global Debt Dashboard

## Project Overview

This Power BI dashboard analyzes debt levels across countries, debt categories, and historical periods to identify global borrowing trends and economic patterns.

---

## Business Problem

Governments and financial analysts require insights into debt distribution to evaluate fiscal sustainability and identify countries with high debt exposure.

---

## Dataset Information

The dataset contains:

- Country
- Debt Percentage
- Debt Categories
- Historical Debt Records
- Government Debt
- Household Debt
- Corporate Debt

---

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Data Modeling

---

## Dashboard Features

- KPI Cards
- Debt by Decade
- Debt Trend Analysis
- Debt by Country
- Highest Debt Categories
- Top Countries by Debt

---

## Key Insights

- Average global debt reached approximately 51.63%.
- Cyprus recorded the highest debt level.
- Corporate debt represented the largest debt category.
- Global debt has steadily increased over recent decades.
- Government debt remains a major contributor worldwide.

---

## Business Recommendations

- Monitor countries with rapidly increasing debt.
- Strengthen fiscal sustainability policies.
- Improve debt management frameworks.
- Encourage responsible public borrowing.
- Enhance transparency in debt reporting.
